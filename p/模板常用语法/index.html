模板常用语法

Aug 16, 2024
阅读时长: 5 分钟

文本

通过{{}}可以将Vue的数据插入网页中

通过v-once可以让其中的数据只进行一次渲染，不会随着数据更新而更改，如<p v-once>{{person.name}}</p>。

显示原生 HTML 代码

通过v-html=varname实现将变量varname当作html代码来渲染。

条件判断

模板可以根据条件选择进行渲染，提供的语法为：v-if、v-else-if、v-else组合实现条件判断。

模板也提供v-show=”true“选择是否渲染内容。

v-show与v-if的区别在于是否加载到网页中，v-show的内容无论是否渲染，都加载到网页中，只是不渲染而已；v-if只有在第一次条件为真时，才会加载到网页中。

循环

在模板中可以使用v-for实现对Vue中的对象、数组进行遍历等操作。

对于数组 

1
2
3
4
5
6

<!-- book为数组遍历的返回，index为索引号（0开始），变量名均可自定义 -->
<tr v-for="(book, index) in books">
    <td>{{ index }}</td>
    <td>{{ book.title }}</td>
    <td>{{ book.author }}</td>
</tr>

对于对象 

1

<div v-for="(value, key) in books">{{ key }}:{{ value }}</div>

保持一致

通过循环得到的HTML标签，在没有使用key属性标记时，若是数据发生改变（如顺序改变），默认标签会重用，只是数据会改变，即HTML代码不变，只有{{}}中的内容会改变，但在同一循环的input等标签不会随着改变。

若想保持一致，如下：

1
2
3
4
5

<!-- 循环所产生的 HTML 标签绑定到对应的 key 上-->
<div v-for="(book, index) in books" v-bind:key="book.name">
    <label for="book">{{ book }}</label>
    <input type="text" v-bind:placeholder="book" />
</div>

会触发视图更新的数组操作

  • push()：添加元素的方法
  • pop()：删除数组最后一个元素
  • shift()：删除数组的第一个元素
  • unshift(item)：在数组的开头位置添加一个元素
  • splice(index,amount,item1,...,itemX)：向数组中index删除amount个元素，添加元素item1,...,itemX
  • sort(function(x,y))：排序，自定义一个function，若运算后返回不小于 0，则x在前，反之y在前
  • reverse()：将数组元素进行反转

<script setup name="App">
    import {  ref } from "vue"

    let count = ref(0);

    const subtract = function (value){
      count.value -= value;
    }

    const getevent = function (value,event){
      const.value += value;
      console.log(event)
    }
</script>

<template>
    <p>{{ count }}</p>
    <!-- 通过 v-on 绑定事件：count += 1 -->
    <button v-on:click="count += 1">加 1</button>
    <!-- 使用 @事件 代替 v-on:事件 来绑定事件：subtract(10) -->
    <button @click="subtract(10)">减10</button>
    <!-- 使用固定的 $event 传入事件的属性 -->
    <button @click="getevent (10,$event)">减10</button>
</template>

事件修饰符

在绑定事件中，我们可以在绑定的函数中对事件进行操作，如阻止超链接的点击默认跳转，就可以在超链接绑定的函数中加入：

1
2

// 需要传入 $event
event.preventDefault();

也可以采用更加简略的写法，直接在绑定时就实现（事件.prevent）：

1

<a href="https://www.baidu.com/" v-on:click.prevent="gotoWebsite($event)">百度</a>

常见修饰符有：

    1. .stop：event.stopPropagation，阻止事件冒泡（事件发生后，会向外层标签发生通知，这就是冒泡）
    2. .capture：事件捕获
    3. .once：这个事件只执行一次
    4. .self：代表当前这个被点击的元素自身
    5. .passive：在页面滚动的时候告诉浏览器不会阻止默认的行为，从而让滚动更加顺畅

</select>
<p>分类为：{{category}}</p>
</div>
</template>
